# 鉄道研究部部規則V1.10
## 序文
本則は、企画の成立の過程、役職に任命や設置に関する条件、買い出しの過程や予算の承認の過程、部員の心得を明示的に定義することで、より円滑に鉄道研究部を運営・活動を行うことを目的として制作したものである。


## 第1章 総則
### 第1条（目的）
本部は、鉄道に関する研究、調査、情報共有及び、模型製作を通じて、部員の知識と技術を深め、それを基に鉄道の魅力を発信することを目的とする。
### 第2条（名称）
本部は、芝浦工業大学附属中学高等学校鉄道研究部と称する。
### 第3条（設立日）
本部の設立日は、yyyy年mm月dd日とする。
### 第4条（所在地）
1. 本部は、芝浦工業大学附属中学高等学校生徒会が定めるところの部である。
2. 部室は、学校が指定する場所に置く。
### 第5条（責任の所在）
本部の運営は、部長を中心とする役員によって決定され、部長と監査の責任の下で行われる。ただし、役員とは、組織図に記載されているすべての役職に就く人または団体・会議を指す。
### 第6条（基準）
本部は生徒会会則を最高規則とする。それゆえ、生徒会会則と本部則の間で齟齬が生じた場合は、矛盾等が生じないように1週間以内に改正しなければならない。ただし、考査前など本部の活動の頻度が極端に低い場合は活動再開日より1週間以内に改正しなければならない。

## 第2章　役職
### 第7条（任期）
任期は4月1日から翌年3月31日までを1年とし、再任は連続2期を超えない。ただし、高2の部員数が極端に少ない場合はこの限りではない。
### 第8条（部長・副部長）
幹部の互選により、部長と副部長を指名し、顧問が任命する。ただし、両役職を兼任することはできない。
### 第9条（会計）
1. 会計担当者を置き、部の財務を管理する。
2. 会計担当者は幹部の互選により選出する。
3. 部長との兼任はできない。
### 第10条（監査）
1. 独立した監査担当者を置き、部の会計及び活動を半期に1回以上監査する。
2. 監査結果は部長及び役員会に報告しなければならない。
3. 会計との兼任はできない。
### 第11条（組織の新設）
1. 上記に加えて班や課などの下部組織を追加で設置することができる。
2. 組織は鉄道研究部＞課＞班＞係の順に定義される。
3. 班や課など設置する場合は少なくとも課長、班長などの長を一人選出すること。
4. 部長及び監査は、部員及び部の機能についての監査義務があり、業務をこなしているかの監査を行う。
### 第12条（引き継ぎ）
毎年1月から3月を、翌年度4月からに向けて知識や技術の共有を行いながら次期幹部が実務経験を積むことを目的とした引き継ぎ期間と定める。ただし今年度の幹部（以下、現幹部とする）の責任のもと運営を行い、問題が発生した際は現幹部の指導・責任のもと対処する。
### 第13条（定例会）
各活動終了時には定例会を開催する。司会は部長とする。参加者は出席部員全員とする。ただし、部長が不在の場合は幹部の中から選出した代理人が務める。
### 第14条（臨時会）
臨時会は部長又は顧問がいつでも開催することができる。参加者は出席部員全員とする。開催の旨はこれをClassiグループ「鉄道研究部20XX（XXには年度が入る）」にて周知しなければならない。

## 第3章 物品・買い出し
### 第15条（備品）
備品(部の予算で購入した物品)を破損した場合は、弁償しなければならない。
### 第16条（私物）
私物（部員が自身の費用で購入した物品）は、原則として自己管理とする。万が一破損等のトラブルが発生した場合、鉄道研究部及び学校側はいかなる対処も行わない。
### 第17条（買い出し）
1. 年間の買い出し全体予算は、臨時会で承認する。
2. 予算の管理は会計担当者が行う。
3. 買い出しの日程は役員会で部長が決定し、定例会で発表する。
4. すべての買い出しは、顧問または副顧問の事前承認を必要とする。
5. 買い出しを希望する品物がある場合は会計担当者に提出すること。口頭での伝達等その他手段は一切認めない。
6. 買い出しリスト（以下リスト）には必ず以下の内容を記載すること。
・商品のジャンル（紙類、木材など）
・店舗名（店の正式名称を記載する）
・商品名
・単価
・税率
・数量
7. 買い出しには活動内買い出し、活動外買い出しの2種類がある。
### 第18条（活動内買い出し）
1. 活動内で買い出しを行う場合は、各班長が会計担当者リストを提出する。その後、会計担当者が集約し、買い出しの2日前までに顧問または副顧問に提出し、承認を得る。
2. 買い出し当日は、顧問または副顧問の外出承認を得た上で会計担当者が責任者となって行う。会計担当者が不在の場合は部長または会計担当者が他の人を指名する。
3. 買い出しの際は必ずレシートと領収証を受け取り、宛名欄に「芝浦工業大学附属中学高等学校鉄道研究部」または「芝浦工大附属中高鉄道研究部」と記載した上で、顧問または副顧問に提出する。その際、お釣りに過不足がないかを必ず確認する。
4. 学校に戻り次第会計担当者が領収書を写真撮影し、ファイルをClassiのコンテンツボックスにアップロードする。

### 第19条　（活動外買い出し）
1. 不測の事態で材料が不足し追加で買い出しが必要になった場合、又は全体で買い出しを行える店舗で品物を扱っていない場合にのみ行うことができる。その際の立て替えは一切認めない。
2. 活動外で買い出しを行う場合は、個人が会計担当者経由で顧問又は副顧問にリストを提出する。その後、予算受取日の活動終了1時間前までに顧問又は副顧問に提出し、承認を得る。
3. 予算受取日の活動終了時に顧問又は副顧問から予算を受け取る。責任者は、高校生の部員1名とする。
4. 買い出しの際は必ずレシートと領収証を受け取り、宛名欄に「芝浦工業大学附属中学高等学校鉄道研究部」または「芝浦工大附属中高鉄道研究部」と記載した上で、翌活動日に顧問または副顧問に提出する。
4. 翌活動日に会計担当者が領収書を写真撮影し、ファイルをClassiのコンテンツボックスにアップロードする。


## 第4章 企画
### 第20条（立案と実行）
出場する大会や、活動で行うことは自由とするが、公序良俗に反さず、鉄道研究部の活動理念に基づくものに限る。ただし、企画書の提出を義務づける。

### 第21条（企画書）
企画書は以下の項目を必ず入れ、期限までに部長の承認を得た上で顧問に提出すること。ただし、企画書の提出期限や提出形式（紙媒体、PDFなど）は部長が決定するものとする。
又、必ず以下の内容を入れること。
・予算と購入物品
・制作場所
・作業目的
・作業内容
・制作者
顧問の承認があり次第企画を行う。

## 第5章　部員　
### 第22条（出欠席）
部員は可能な限り部活動に参加する義務がある。また、欠席・遅刻・早退の連絡は緊急の場合を除き、活動開始前までにClassiグループ「鉄道研究部20XX（XXには年度が入る）」の欠席連絡投稿にコメントする。

### 第23条（心得）
第1条の目的を達成するために、部員は鉄道研究部の活動に協力する義務がある。
いかなる時も優先座席には座らないこと。
予算で活動していることを自覚し、日々精進する。

## 第6章　改正
### 第24条（改正に関する手続き）
本則の改正には、臨時会にて出席部員の3分の2以上の賛成を必要とする。ただし、第6条に則った改正の場合はこの限りではない。又改正の場合は公布日・施行日をClassiグループ「鉄道研究部20XX（XXには年度が入る）」にて通達する。
