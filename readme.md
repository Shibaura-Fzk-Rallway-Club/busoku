## 序文
本則は、企画の成立の過程、役職の任命・設置に関する条件、予算の承認・買い出しの過程、部員の心得を明確に定義することで、より円滑に鉄道研究部の運営・活動を行うことを目的として制作したものである。

## 第1章 総則
### 第1条（活動目的）
本部は、鉄道に関する研究、調査、情報共有及び、模型製作を通じて、部員の知識と技術を深め、それを基に鉄道の魅力を発信することを目的とする。
### 第2条（名称）
本部は、芝浦工業大学附属中学高等学校鉄道研究部と称する。
### 第3条（所在地）
1. 本部は、芝浦工業大学附属中学高等学校生徒会が定めるところの部である。
2. 部室は、学校が指定する場所に置く。
### 第4条（責任の所在）
本部の運営は、部長を中心とする役員によって決定され、部長と監査の責任の下で行われる。
### 第5条（基準）
本部は生徒会会則を最上位規約とする。それゆえ、生徒会会則と本部則の間で齟齬が生じた場合は、矛盾が生じないように遅滞なく改正しなければ生徒会会則違反となる。

## 第2章　役職
### 第6条（役員・幹部）
1. 役員は部長、副部長、会計に加えて部長が部員から選出した者によって構成される
2. 選出された役員は年度始めの臨時会で部員全体に発表する義務がある
3. 幹部とは、高校2年次の部員全員を指す。
### 第7条（任期）
任期は4月1日から翌年3月31日までを1期とし、再任は連続2期を超えない。但し、幹部の人数が極端に少ない場合はこの限りではない。
### 第8条（部長・副部長）
幹部の互選により、部長と副部長を指名し、顧問が任命する。但し、両役職を兼任することはできない。
### 第9条（会計）
1. 会計担当者を置き、部の財務を管理する。
2. 会計担当者は幹部の互選により選出する。
3. 部長との兼任はできない。
### 第10条（監査）
1. 幹部から独立した監査担当者を置く。
2. 監査担当者は高校1年次(幹部の1学年下の代)の全員を含めるものとする。但し、幹部は監査担当者との兼任はできない。
3. 監査担当者は、毎年10月と3月に会計の監査を行い、予算使用状況と領収書の金額が一致していることを確認し、結果をまとめた報告書を作成し、顧問に提出しなければならない。尚、10月は本予算及び芝生祭予算、3月は本予算及び補正予算の監査をそれぞれ行う。
### 第11条（組織）
1. 部活動の能率的な遂行のために必要な部内組織を整えることを目的とし、本部の統轄の下における組織は、その業務について、自ら評価し、企画及び立案を行い、並びに本部の組織相互の調整を図るとともに、その相互の連絡を図り、全て、一体として、組織機能を発揮するようにしなければならない。
2. 本部における組織の順位は、上から鉄道研究部、課、班、係の順に定義される。
3. 本部の統括の下における組織を設置する場合は、各組織ごとに長を一人選出する。
4. 部長及び監査は、部員及び部の機能についての監査義務があり、業務が能率的に遂行されているかの監査を行う。
5. 組織の新設や廃止は部長の承認のもと行う。
### 第12条（引き継ぎ）
毎年1月から3月を引き継ぎ期間と定める。目的は翌年度4月からに向けて知識や技術の共有を行いながら、次期幹部が実務経験を積むことである。但し、今年度の幹部（以下、現幹部とする）の責任のもと運営を行い、問題が発生した際は現幹部の指導・責任のもと対処する。
### 第13条（定例会）
活動時間終了時には定例会を開催する。必ず司会を幹部の中から選出する。参加者は開催時に部活動に出席している部員全員とする。
### 第14条（臨時会）
臨時会は幹部又は顧問がいつでも開催することができる。参加者は開催日時に部活動に出席している部員全員とする。開催の旨はこれをClassiグループ「鉄道研究部20XX（XXには年度が入る）」にて周知しなければならない。

### 第15条（役員会・幹部会）
1. 役員会は、役員により構成される。役員は役員会への出席義務、および同会における議決権、役員以外の招待権を与えられる。なお、役員の議決権の棄権、および他人への譲渡はこれを認めない。
2. 議題は部長、副部長双方が賛成し、かつ役員全員の3分の2以上の賛成で可決される。
3. 役員による役員以外の招待権の発動は、少なくともいずれか1人の役員の承認により行うことができ、参考人として役員以外の部員の招待を行える。ただし、当該部員は役員会における発言権のみ認められる。
4. 幹部会は幹部全員により構成される。
5. 幹部会への幹部以外の招待はこれを認めない。
6. 幹部は幹部会への出席を義務づけ、同会における議決権を持つ。なお、幹部の議決権の棄権、および他人への譲渡はこれを認めない。
7. 議題は部長、副部長双方が賛成し、かつ幹部全員の3分の2以上の賛成で可決される。
## 第3章 物品・買い出し
### 第16条（備品）
備品(部の予算で購入した物品)を破損した場合は、破損者がそれを弁償しなければならない。
### 第17条（私物）
私物（部員が自身の費用で購入した物品）は、原則として自己管理とする。万が一破損等のトラブルが発生した場合、鉄道研究部及び学校側はいかなる対処も行わない。
### 第18条（買い出し）
1. 年間の各企画への本予算および補正予算の配分は、予算を必要とする各企画の代表者が予算案を会計担当者に提出の上で、部長が承認したものを顧問に提出する。
2. 予算の管理は会計担当者が行う。各企画に配分された予算額を超えた買い出しを行う場合、会計担当者は買い出しリストに記載の物品を取り下げることができる。
3. 買い出しの日程は会計担当者が決定し、定例会で会計担当者が発表する。
4. 部の予算で行う全ての買い出しは、顧問又は副顧問への買い出しリストの提出とその承認を必要とする。
なお、買い出しリスト（以下リスト）には必ず以下の内容を記載すること。
・商品のジャンル（紙類、木材など）
・店舗名（店の正式名称を記載する）
・商品名
・単価
・税率
・数量
7. 買い出しには活動内買い出し、活動外買い出しの2種類がある。
### 第19条（活動内買い出し）
1. 活動内で買い出しを行う場合は、各企画の代表者が会計担当者にリストを提出する。その後会計担当者が集約し、生徒会の定める所定の手続きに従う。
2. 買い出し当日は、顧問または副顧問の外出承認を得た上で会計担当者が責任者となって行う。但し、会計担当者が不在の場合は部長又は会計担当者が他の人を指名する。
3. 買い出しの際は必ず領収証などの品名や金額がわかるものを受け取り、宛名欄に「芝浦工業大学附属中学高等学校鉄道研究部」又は「芝浦工大附属中高鉄道研究部」と記載した上で、顧問または副顧問に提出する。その際、お釣りに過不足が無いかを必ず確認する。
4. 学校に戻り次第会計担当者が領収書を印字内容が確認できるように写真撮影し、年度末まで保管する。
### 第20条　（活動外買い出し）
1. 不測の事態で材料が不足し追加で買い出しが必要になった場合、又は全体で買い出しを行える店舗で品物を扱っていない場合にのみ行うことができる。
2. 活動外で買い出しを行う場合は、個人が部長または会計担当者経由で買い出しに必要な代金受取日の活動終了1時間前までに顧問又は副顧問にリストを提出する。
3. 買い出しに必要な代金受取日の活動終了時に顧問又は副顧問から買い出しに必要な代金を受け取る。責任者は高校生の部員1名とし、予算の過不足、提出したリストの通りに買い出しが行われていることを確認する。
4. 買い出しの際は必ず領収証を受け取り、宛名欄に「芝浦工業大学附属中学高等学校鉄道研究部」又は「芝浦工大附属中高鉄道研究部」と購入店舗係員の記載を受けた上で、翌活動日に顧問又は副顧問に提出する。
5. 値上げ、セール等でリスト通りに買い出しを行った上で、金額に過不足が生じた場合は、提出時までに買い出しリストを更新し、領収証と一緒に提出する。なお、提出期限は翌活動終了1時間前とする。
6. 翌活動日に会計担当者が領収書を印字内容が確認できるように写真撮影し、年度末まで保管する。
### 第21条（インターネットを用いた買い出し）
1. 実店舗ではなくインターネットを用いて購入する場合は、都度各企画代表者が会計担当者経由で顧問又は副顧問にリストを提出する。記載する内容は、第18条の4項に記載されているものに加えて購入するサイトのリンクとする。
2. 会計担当者は、インターネットを用いた買い出しで使った金額を記録すること。また、会計担当者が領収書を印字内容が確認できるように写真撮影し、年度末まで保管する。

## 第4章 企画
### 第22条（企画書）
出場する大会や、活動で行うことは自由とするが、公序良俗に反さず、本則第1条の活動目的に関連したものに限る。但し、企画書の提出を義務付ける。
幹部が決定した、企画書の提出期限や提出形式（紙媒体、PDFなど）に従い、部長の承認を得た上で顧問に提出すること。但し、必ず以下の内容を入れること。
・大会等までの作業計画、作業内容
・予算と購入物品
・制作場所
・作業目的
・制作者
顧問の承認があり次第企画を行う。

## 第5章　部員　
### 第23条（出欠席）
部員は可能な限り部活動に参加する義務がある。又、欠席・遅刻・早退の連絡は緊急の場合を除き、活動開始前までにClassiグループ「鉄道研究部20XX（XXには年度が入る）」の欠席連絡投稿にその旨とその理由をコメントする。

### 第24条（心得）
第1条の目的を達成するために、部員は鉄道研究部の活動に協力する義務がある。
予算で活動していることを自覚し、日々精進する。

## 第6章　改正
### 第25条（改正に関する手続き）
本則の改正には、臨時会にて総部員の3分の2以上の賛成を必要とする。但し、その際は総部員の3分の2以上が出席している必要がある。万が一3分の2に満たなかった場合は、別日に臨時会を開催する。また、第6条に則った改正の場合はこの限りではない。
又、改正の場合は公布日・施行日をClassiグループ「鉄道研究部20XX（XXには年度が入る）」にて通達する。
